Book of Mormon Tree Chart Help Page

Getting Started

The Book of Mormon Tree graph is divided into three layers which can be navigated by the user

The three layers are as follows from most general to specific:

- Word Type: The entirety of the data set is on display, with highlight colors distinguishing the function words from content words. This provides a large scale overview of the prominence of function vs. content words for the current data set.

- Part of Speech: Within each set of either content or function words, the next layer breaks down either set by part of speech. This gives a better picture what exact type of words are most common within the data set, giving a more detailed picture.

- Individual Words: Within each break down of a set of part of speech words, individual words can be selected and explored. Hovering individual words will give the frequency value and other details of interest. This is the most specific layer of the chart.


Changing Datasets

Each Bubble Chart is created from data for a single speaker from the Book of Mormon text. The sidebar on the right contains the name of each speaker. In order to view the charts for a specific person, the user can click on the name and the graphs will automatically be redrawn. The currently selected dataset/speaker will be highlighted in bold text for clarity.

Chart Interaction

While viewing the chart, has a few options for interacting with the graph. If a section of the chart is hovered with the cursor, the exact frequency value of the word (how many times it appears in the text) will be displayed.

If a section is clicked, the graph will "drill-down" on the selected section, redrawing itself to fill the chart display area with only the selected data. (Ex. If the 'Content Word' half of the chart is selected from the inital display, the function words half will vanish and the content word half will expand and reveal the breakdown by part of speech.)
